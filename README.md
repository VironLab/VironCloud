## Maven Dependency

```gradle

//Repository - Groovy
maven {
    name = "vironlab"
    url = "https://repo.vironlab.eu/repository/snapshot/"
}

//Repository - Kotlin DSL

//Dependency - API (All Enviroments)

//Dependency - Core (Only Extension)

```
